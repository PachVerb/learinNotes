# 自定义模块
 - require() 引入模块
 - exports 输出
 - moudle 模块

引入自己得模块，必须在模块前加 ./

>nodejs模块没有全局变量。原因在于在创建变量时，node语法会在代码块创建立即执行函数表达式。

# 模块内容输出
- 单个输出
```
    exports.a = 12;
```
- 批量输出
```
    module.exports = {
        b:12,
        c:15,
        d:16
    }
```

由于nodejs里文件本身就是js，在引入.js里引入模块时，可以去掉.js后缀。

# 重点
require() 引入的模块查找规则

如果有 ./，直接在当前项目目录查找
如果没有，
    先查找系统模块
    在从node_modules目录里找
